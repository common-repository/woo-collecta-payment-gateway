== Description ==
Collecta Woocommerce Payment Gateway allows payment on your Woocommerce store, It Accepts mastercard, visacard and Verve Card .
**Note**: This plugin Process transaction in Naira Only.


== Installation ==
1. Login to your WordPress Admin area
2. Go to "Plugins > Add New" 3. Search for "Collecta"
4. From the search result you will see "Collecta Woocommerce Payment Gateway" click on "Install Now" 5. Click "Activate Plugin" to activate it.
6. Open the settings page for WooCommerce and click the "Check Out," tab.
7. Click on the sub tab for "collecta".
8. Configure your "Collecta Payment settings. See below for details.


**Configure the plugin**1. Enable/Disable - check the box to enable or disable Collecta Payment Gateway.
2. Title - This is  what your customers will see  on the checkout page.
3. Description - this will appear under the payment fields on the checkout page. Here you can list the types of cards you accept.
4. Handle  - enter toyr collecta handle (without @ symbol) which can be seen when you login to collecta website and click on developer.5 Secret Key - Enter your secret key which will also be seen on your dashboard on collecta website. 6. Click on **Save Changes** for the changes you made to be effected.
